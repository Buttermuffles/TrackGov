1. Start with `document-lifecycle.md` for business flow.
2. Apply `security-hardening.md` controls before production release.
3. Implement optimization tasks from `performance-and-caching.md`.
4. Enforce coding standards in `error-handling-and-resilience.md`.
5. Use `operations-runbook.md` for incidents and recovery.
